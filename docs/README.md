## Quick Start

For developers new to the codebase, we recommend reading in this order:

1. **Product Overview** - understand what Marlin does
2. **Architecture** - understand how components fit together
3. **Data Model** - understand the core entities
4. **Backend API** or **Smart Contracts** - depending on your focus area
