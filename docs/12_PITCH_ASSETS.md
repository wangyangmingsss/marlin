# Pitch Assets

## Demo Video Script Outline

1. **Hook** (10s): "What if Stripe Billing worked with stablecoins on Solana?"
2. **Problem** (20s): Businesses accepting stablecoins lack invoicing, subscriptions, and reconciliation tools.
3. **Dashboard walkthrough** (45s): Create merchant, create invoice, show checkout flow, see payment arrive.
4. **Subscription demo** (30s): Create plan, customer subscribes, automated charge fires.
5. **Developer experience** (20s): Show API call, SDK usage, webhook delivery.
6. **Architecture** (15s): Quick flash of the system diagram — non-custodial, on-chain settlement.
7. **Closing** (10s): "Marlin — stablecoin billing is now."

Total target: ~2.5 minutes.
