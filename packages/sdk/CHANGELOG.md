# Changelog

## 1.0.0 (2026-05-06)

### Features

- Initial release of `@marlinfi/sdk`
- Full TypeScript client for the Marlin billing API
- React integration via `@marlinfi/sdk/react` subpath
- Confidential invoices support via `@marlinfi/sdk/confidential` subpath
- Webhook signature verification utilities
- Solana transaction helpers
